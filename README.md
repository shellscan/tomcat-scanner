# tomcat-scanner
* A package of tools which are used to scan *Tomcat Memory Shell*. It covers the following components:
  1. Service
  2. Engine
  3. Host
  4. Servlet
  5. Listener
  6. Filter
  7. Valve
  8. Connector
  9. WebSocket
* How to use?
  1. Put the tools package into the web root directory.
  2. Access the jsp file by browser.
